How should custom shortcuts (!bangs) work?

Example: typing "!w telescope" to search Wikipedia directly.

A

Silent Shortcuts

Works just like DuckDuckGo. You type "!w" and hit Enter; the system handles the redirect automatically without any visual changes while typing.

B

Visual Feedback (Chip)

As soon as you type "!w", the input field shows a small "Wikipedia" chip or icon to confirm it's switching engines for this search.

Wikipedia
C

Autocomplete List

Typing "!" opens a small list of available shortcuts so you don't have to remember them all.
