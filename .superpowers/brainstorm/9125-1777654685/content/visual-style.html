Choose a visual direction for your homepage

Since you want many search options, let's start with how you'd like them organized.

A

Classic Search Hub

A central, familiar search bar like Google. Quick-switch buttons for other engines (Bing, DuckDuckGo, etc.) appear right below the search box.

B

Power User Dashboard

Grid-based layout. Shows your most used engines as cards, or a sidebar that lets you quickly toggle between search providers before you type.

C

Modern Minimalist

Ultra-clean layout. A single input field that automatically detects the engine via shortcuts (e.g., "!g" for Google, "!y" for YouTube) or a subtle subtle engine indicator.
